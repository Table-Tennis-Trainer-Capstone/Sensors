# TABLE TENNIS TRAINER
## Spring 2026 Capstone Project

A high-speed, vision-guided robotic system designed to autonomously engage in a Table Tennis match with a human opponent. This project utilizes a distributed ROS2 architecture across two NVIDIA Jetson Nanos to handle GPU-accelerated vision processing and high-frequency arm control.
